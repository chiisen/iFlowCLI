# iFlowCLI ✨

iFlow CLI (命令列界面) 專案 🌟  
iFlow 是一款 AI 驅動的開發工具，專為提升開發效率而設計 🚀

![iflow](./images/iflow.png)

## 官網 🌐

### 技術文件 📚
https://platform.iflow.cn/cli/quickstart/index.html

### 線上客服 💬
https://iflow.cn/

## 安裝 💻

**環境需求**  
- ⚠️ Windows 系統  
- 📦 Node.js 22+  

```bash
npm install -g @iflow-ai/iflow-cli@latest
```

## 啟動 🚀

```bash
iflow

iflow --version
```
